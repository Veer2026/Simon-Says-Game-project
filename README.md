# Simon-Says-Game-project
Simon says is a memory game where we follow a pattern and remember color name in a ordered series . If we choose any wrong color in series we will be out.
</br>
<b>Instruction to play :-</b>
First we press any key from keyboard to start the game, once we press key the game will start and blink a color we have to click on this color box for next level and remember it. In next level an another color box will blink randomly then we have to click first level color and second level color, the game will be continue untill we miss any color from series.
<br>
For example we play first level and the color was red then we click on red and move second level, in second level the color blinked was yellow then we click Red and yellow in order to move next level if we miss the color order we will be out and game will over with a highest score.
